Follow these Steps to run Amstat_Aircraft or Amstat_Market


1. Run "Amstat_Aircraft_runner.bat" file as administrator.
2. Run "Amstat_Market_runner.bat" file as administrator.

		OR

STEP -1 : Open a Command Prompt or Terminal:

	- Open a command prompt (Windows) or terminal (macOS/Linux) window.
	- Navigate to the Project Directory:
	- Use the cd command to navigate to the top-level directory of the Scrapy project.

	"cd path\to\your\scrapy\project"


STEP - 2 : Run the Scrapy spider using the following command:

	"scrapy crawl Amstat_Aircraft"
		  or
	"scrapy crawl Amstat_Market_Scrapy"
	            Respetively
